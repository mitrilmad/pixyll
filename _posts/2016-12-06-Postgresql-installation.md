* sudo yum install postgresql
* postgresql 을 yum으로 설치하면 자동으로 postgre 계정 생성됨
* postgre 계정 패스워드 부여 및 로그인
* -bash-4.2$ initdb
* -bash-4.2$ pg_ctl start
* postgres=# Create database playdb;
* postgres=# select * from pg_database;
